Project Description:

The Online University Management System is an application which will provide easy and flexible assistance to the educational institutes to manage the data and functions for their students as well as faculty. It will provide many functions which will make it easier for teachers and administrators to manage their institutes successfully. The management system will be implemented on a website that will support an easy-to-use UI/UX. The project will use suitable frameworks for HTML, CSS and JS for designing a responsive frontend. MySQL database will be used to manage the data

Project Features:

  • Student attendence
  • Maintain test scores
  • Maintain student information
  • Online course registration
  • Maintain student assignment
  • Announcements
  • Upload course materials
  • Contact administration
  • Contact student and teachers
  • Calculate and display fee
  • Online course registration
  • Student study plan
  • Course feedback
  • UOnline society registration
