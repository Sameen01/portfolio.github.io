Steps for hosting:

5) Click "Start a project".

7) As a quick way to create the first file, click the "README" link
8) for a file named README.md. Change the filename from README.md to index.html.
9) Delete the current contents of the file editor and replace them with the HTML of your webpage.
10) Scroll down the page until you find the Commit area, and click “Commit new file”.
11) Type YOUR_USERNAME.github.io in the browser, and reload until your webpage appears.
